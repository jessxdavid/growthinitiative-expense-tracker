# Changelog

All notable changes are documented here. Dates are YYYY-MM-DD.

## [1.0.0] - 2026-07-23
Initial public template.

### Added
- Expense + income logging with an Expense/Income toggle.
- Automatic team profit-share: profit = income minus expenses, each member paid their % of profit, mark-paid per period.
- Custom categories with monthly budget caps.
- Business / Personal tag with filtering.
- Recurring items that auto-add each month.
- Attachments (drag-drop, click, or paste) with inline preview + lightbox.
- Dashboard: Total Spent / Income In / Profit / Payouts Owed tiles, insights, monthly trend, category donut, spend-by-name.
- Date ranges: this week / this month / all time / custom.
- CSV export and a printable PDF report.
- JSON backup / restore.
- Optional live cloud sync across devices via a Cloudflare Worker + KV.
- Demo mode (`?demo`) that loads sample data for previews.
